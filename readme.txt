Markup language:
Wikicreole

Description:
A basic Wikicreole markup set with Headings, **Bold**, //Italic//, Link, Picture, Lists, Quotes and Codes.

Install:
* Download the zip file
* Unzip it in your markItUp! sets folder
* Modify your JS link to point at this set.js
* Modify your CSS link to point at this style.css
